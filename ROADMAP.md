# Roadmap

This roadmap reflects what we're planning. Priorities may shift based on user feedback.

Upvote features by reacting 👍 to the relevant issue. The most-requested items move up the list.

---

## In Progress

Nothing currently in active development beyond v1.0 launch.

---

## Planned

### Mosh Support
UDP-based connection protocol for mobile networks and high-latency links. Requires cross-compiling the mosh-client as an XCFramework for iOS — significant engineering effort.

### Smart Command Suggestions
Frequency-based command completion in the keyboard accessory bar, surfacing your most-used commands as quick-tap suggestions.

---

## Under Consideration

These are ideas we're evaluating based on demand. Open a feature request if something here matters to you.

- **SSH Agent Forwarding** — Forward your local SSH keys to remote hosts for chain authentication
- **Native Split Panes** — Side-by-side terminals without requiring tmux
- **Log Viewer** — Tail and filter log files with native UI
- **Widget / Lock Screen widget** — Quick-connect from the home screen
- **iPad multitasking** — Stage Manager and Split View optimization

---

## Not Planned

- **Local shell** — iOS sandbox restrictions make a fully functional local shell impractical
- **macOS support** — The app is designed specifically for touch + mobile workflows
